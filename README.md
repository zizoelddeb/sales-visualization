# Sales Visualization Project

This project analyzes and cleans sales data using MySQL.

## Dataset
Online Retail II Dataset

## Table Used
online_retali_raw

## Data Cleaning
- Removed rows with missing customer IDs
- Removed returned orders (negative quantity)
- Created TotalSales column

## Analysis
- Top 5 products by revenue
- Top 3 countries by revenue (excluding UK)
- Orders distribution by hour

## Tools
- MySQL
